git terms/command

git clone - выгрузить проект через HTTPS or SSh(рекоменд. через него)

git status - статус проекта, просмотр всех манипуляции с файлами

git add - добавить файл в очередь, следить за файлами

git commit - создать версию проекта, скриншот состояния проекта

git push - заливать в гитхаб все локаль.файлы

git remote - посмотреть ссылку на наш проект

git barnch zhan - создаёт ветку "zhan"

git checkout - переходить между ветками, не забвть написать имя ветки

RULE: нельзя писать коммиты в прошлом времени, нужно написать как что-то происходит
